# Dashboard E-commerce

Panel de control para gestión de e-commerce construido con HTML semántico, CSS moderno y JavaScript vanilla. Consume datos reales de FakeStore API.

## Funciones
- KPIs en tiempo real — ventas, productos, categorías y precio promedio
- Gráficas interactivas — ventas por categoría y distribución de precios
- Tabla de inventario con búsqueda y filtro por categoría
- Tema oscuro/claro con persistencia
- Diseño responsive con sidebar drawer en móvil
- Accesibilidad — ARIA, skip link, live regions

## Tecnologías
- HTML5 semántico y accesible
- CSS — BEM, @layer, Custom Properties, Grid + Flexbox
- JavaScript ES Modules — sin frameworks ni bundlers
- Chart.js — gráficas interactivas
- FakeStore API — datos reales

## Aprendizajes
- Arquitectura CSS escalable con @layer y design tokens
- JavaScript modular sin frameworks
- Accesibilidad web — WCAG, ARIA, navegación por teclado
- Sanitización XSS con DOM API
- Responsive design con CSS Grid y media queries
